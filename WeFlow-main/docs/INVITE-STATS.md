# 邀请统计口径

## 数据大屏邀请人数排行榜

- 默认不开“去重”时，排行榜按群成员邀请溯源中的邀请记录逐条计数。
- 同一个邀请人下，只要被邀请人记录存在于邀请溯源中，就计 `1` 条；默认不去重时按溯源表展示的邀请者名称聚合。
- 默认不因状态、归因有效性或是否退群而排除记录。
- 排行榜只统计邀请类溯源记录；单独退群事件不进入邀请人数排行榜。
- 打开“去重”后，沿用原有去重逻辑：按成员身份去重，并使用原有有效邀请口径。

## 展示文案

- 数据大屏“邀请人数排行榜”描述使用：
  `【活动名】招募者 xx 名，总人数 xxx`
- `总人数` 使用同屏“总成员数”卡片当前显示值。
- 下载排行榜图片的标题使用同一口径。

## 群成员溯源

- Web 端群成员溯源搜索保持“成员昵称”搜索语义。
- 成员昵称匹配会去除首尾空白、压缩连续空白、忽略大小写，并移除零宽字符，避免表格中存在可见名称但筛选数量偏少。
