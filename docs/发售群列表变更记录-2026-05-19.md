# 发售群列表变更记录（2026-05-19）

## 完成内容
- 在 `WeFlow-Web` 单页壳中新增 `发售群列表` 视图，顶部视图变为 `数据大屏 / 发售群列表 / 群成员溯源`
- 新增发售群列表展示：
  - 当前活动标签下的群列表
  - `序号 / 群名称 / 人数 / 操作`
  - `包含已退群的人` 开关
  - 群数量与人数摘要
- 新增四个导出入口：
  - `合计导出群人数`：CSV
  - `批量导出所有群员`：ZIP，按群拆分 CSV
  - `合计导出群列表`：CSV
  - 行内 `导出群员`：单群 CSV
- `包含已退群的人` 只影响顶部总人数口径，不改变表格行内人数
- 更新 README，将 `发售群列表` 从“不包含”改为已包含能力
